## Installation

Clone the repository `git clone https://github.com/WebExp0528/Email-Validation-Extension.git`

#### Run backend

1. Run `cd backend+dashboard`
2. Run `npm start`

#### Run extension

1. Run `cd extension`
2. Run `npm install`
3. Run `npm run build` or `npm run watch-dev`
